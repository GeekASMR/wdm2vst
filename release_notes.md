## 🚀 WDM2VST V3.2.1 稳定版正式发布 

### 🎵 音频体验修复
- **解决听感重音 Bug**：彻底修复了在宿主软件 (DAW) 开启输出桥接时，底层电脑喇叭还会跟着一起发声，导致听到重复“双重声音”的老大难问题。
- **更强的爆音排查能力**：底层驱动加入了极低损耗的诊断记录。现在如果因为系统“采样率不匹配”等原因遇到滋滋滋的爆音，配合排查解决起来将更加精准迅速。

### 📦 更纯净的安装体验
- **纯净无痕安装**：全新优化的安装包机制。驱动一旦安装成功，所有刚才使用过的临时的配置脚本和无用文件都会立刻“阅后即焚”，从此不留多余垃圾，还你一个极致干净的驱动目录。
- **精简插件架构**：剔除了旧版本产生的一些闲杂多余文件夹，现在的文件更加轻巧纯粹，从根源上消灭了以前由残留垃圾引起的莫名崩溃闪退。

### 🛡️ 新手避坑防护
- **版本更新自动感应**：内置了版本更新雷达，一旦有新版本发布，网页会亮起极客风格的“呼吸霓虹灯”按钮提醒并一键带你更新。
- **小白防挂指南 (报错37 / 52)**：原生加入图形化报错诊断。如果最新版 Windows 因为开启了“内存完整性”拦截了我们的驱动，不再让你两眼一黑！安装器会直接弹出直白的图文弹窗，手把手教你关闭拦截设置来解决问题。
