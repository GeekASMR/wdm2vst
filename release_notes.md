## 🚀 WDM2VST V3.1.1 稳定版 正式发布 

### 🌟 新增核心功能 (New Features)
- **版本高亮更新检测 (Auto Update Checker)**：内核注入全新异步版本校验网络协议。当检测到云端发布新版本时，插件 UI 的右侧将自动展开，呈现独家调校的呼吸霓虹灯（Neon Breathing）按钮，一键全自动跳转极客官网更新。
- **底层遥测诊断哨兵 (Telemetry System)**：增加最深层的异常抛出与参数（SR / BufferSize）监控报告。针对过往由于不同 DAW 采样率不匹配导致的各类鬼畜爆音/卡死问题，现在可通过脱敏日志矩阵实现远程极速定位。*(安装页面含可选项，严格遵循许可下收集)*。
- **热启动智能连接 (Instant Auto-Connect)**：根除了首次加载插件无声、重新打开工程输出断连 (- None -)的痛点。底层 C++ 构造器现在执行光速预判，自动握手第一根可用的 IPC 虚拟骨干通道！

### �� 稳定性修复 (Under The Hood)
- **VST2WDM 异步心跳自愈 (Heartbeat Reconnect)**：在插件定时器中植入了毫秒级静默侦测，即使声卡被外部程序劫持导致共享内存断链，也能在后台0感知重连。
- **证书环境物理洗地 (Registry Firewall Fix)**：针对部分电脑（或虚机）安装后报错驱动不兼容的问题，重置了 Inno Setup 预清理探针。安装前以 PowerShell 的形式直接轰杀系统中残存的孤儿 OEM 配置文件与伪签名，保障 100% 装机即用。
- **自动脱离本地依赖**：打包分发流程全链革新！舍弃附带多余文本，安装向导最终步骤自动弹出 https://geek.asmrtop.cn/wdm2vst.html 在线官网主页，与云端服务浑然一体。
